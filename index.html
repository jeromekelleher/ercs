Extinction/recolonisation coalescent simulator

Simulates the coalescent for populations evolving in a spatial continuum under the extinction/recolonisation model. The simulations support:

  • A sample of n individuals with m loci at arbitrary locations on a torus of diameter L.
  • Recombination occurs between locus j and j + 1 at an event with probability rho[j].
  • An arbitrary number of classes of event occuring at fixed rates.

Ercs supports both Python 2 and 3.

Documentation

Here's a quick example for the impatient::

    import ercs
    sim = ercs.Simulator(10)
    sim.sample =  [(1, 1), (2, 2), (3, 3)]
    sim.event_classes = [ercs.DiscEventClass(rate=1.0, u=0.5, r=1)]
    pi, tau = sim.simulate(1)

Full documentation for ercs is available at <http://www.homepages.ed.ac.uk/jkellehe/ercs/doc>_.
